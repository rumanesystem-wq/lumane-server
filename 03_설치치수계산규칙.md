# 설치 치수 계산 규칙

## 기본 규칙
- 최대 4면까지 합산 가능
- 100mm 단위로 계산
- 소수점 0.5 초과 시 올림
- 기본 행거 단가: 10cm당 9,000원

## 계산 방식
1. 고객이 제공한 각 면의 치수를 mm 단위로 확인
2. 설치 가능한 면만 최대 4면까지 합산
3. 100mm 단위로 정리
4. cm로 환산 후 금액 계산
5. 배송비는 별도 처리

## 예시
- 2400mm + 1800mm = 4200mm
- 4200mm = 420cm
- 420cm ÷ 10cm = 42
- 42 × 9,000원 = 378,000원
