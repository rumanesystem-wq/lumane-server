# 설치 치수 계산 규칙

## 기본 규칙
- 최대 4면까지 합산 가능
- 100mm 단위로 계산
- 소수점 0.5 초과 시 올림
- 기본 행거 단가: 10cm당 10,000원

## 계산 방식
1. 고객이 제공한 각 면의 치수를 mm 단위로 확인
2. 설치 가능한 면만 최대 4면까지 합산
3. 100mm 단위로 정리
4. cm로 환산 후 금액 계산
5. 배송비는 별도 처리

## 예시
- 2400mm + 1800mm = 4200mm
- 4200mm = 420cm
- 420cm ÷ 10cm = 42
- 42 × 10,000원 = 420,000원

## 도면 치수 vs 실측 치수

- 도면 치수로도 예상 견적 안내는 가능하다
- 단, 시공 시에는 반드시 실측 치수가 필요하다
- 도면 치수로 상담 진행 시 반드시 아래 문구를 포함한다:
  "도면 치수로 예상 견적 안내는 가능하지만, 실제 시공 전에는 실측 치수 확인이 필요해요 😊"
- "도면이 더 정확하다"는 표현 절대 금지

## 치수 입력 안내 형식

고객이 치수를 어떻게 입력해야 할지 모를 경우 아래 형식으로 안내한다:

"방문 왼쪽부터 순서대로 알려주시면 돼요 :)
A면:
B면:
C면:
(예: A면: 2400, B면: 1800, C면: 1200)"

- 몰딩
- 기둥
