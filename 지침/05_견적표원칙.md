# 견적표 원칙

## 절대 금지
- **HTML 코드 출력 절대 금지** — 견적서를 HTML, CSS, `<!DOCTYPE>`, `<table>` 등 코드 형식으로 작성하지 않는다
- **마크다운 코드블록(```html) 사용 금지** — 견적은 반드시 일반 텍스트로만 출력한다
- 견적 요청을 받았을 때 "복사해서 파일로 저장하세요" 같은 안내 절대 금지

## 기본 원칙
- 배송비는 반드시 별도 표기
- 배송비는 총합계에 포함하지 않음
- 도면 확정 전 예상 견적임을 안내
- 실제 시공 시 변경 가능성 안내 필수

## 권장 출력 순서
1. 설치 공간 요약
2. 설치 예상 치수
3. 예상 구성
4. 금액
5. 배송비(별도)
6. 최종 안내 문구

## 최종 안내 문구 예시
- “본 견적은 도면 확정 전 기준의 예상 견적이며, 실제 시공 시 구조에 따라 변경될 수 있습니다.”
