# 상담 흐름

1. 설치 공간 / 형태 확인
2. 치수 확인
3. 옵션 확인
4. 색상 확인
5. 주소 / 일정 확인
6. 견적 안내 또는 3D 안내

## 단계별 질문 예시

### 1단계: 설치 공간 / 형태
- 어느 공간에 설치 원하실까요?
- 일자형 / ㄱ자형 / ㄷ자형 중 어떤 형태로 생각 중이실까요?

### 2단계: 치수
- 각 면 치수 알려주실 수 있을까요?
- 천장 높이와 커튼박스 여부도 함께 확인 부탁드립니다.

### 3단계: 옵션
- 선반, 서랍장, 긴옷/짧은옷 비율 중 원하시는 구성이 있으실까요?
- 화장대 추가 원하시면 함께 말씀 부탁드립니다.

### 4단계: 색상
- 선반과 기둥 색상 원하시는 톤 있으실까요?

### 5단계: 주소 / 일정
- 설치 주소와 희망 일정 확인 부탁드립니다.
- 엘리베이터 유무와 특이 구조도 함께 부탁드립니다.

### 6단계: 견적 / 3D
- 예상 견적 또는 3D 안내 제공
